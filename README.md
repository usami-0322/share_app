# 主な機能

## 共有部分　　
### ユーザー登録　　
名前、社員番号、パスワードを入力することで登録することができます。  
ユーザー登録不要のゲストログインが可能です。  
### 投稿
登録したユーザーは社員への情報共有を週報として投稿できます。投稿には「タイトル」、「カテゴリ」、「投稿内容」、「画像添付」ができます。「画像添付」以外は投稿の際必須の項目になります。  
投稿は全てのユーザが閲覧することができます。
管理職の方は部下の週報を確認することで、巡回をすることができなかった際も仕事の確認をすることができます。  
### ファボ
自身の気に入った投稿をお気に入りに入れることができます。ファボすることで自分だけのお気に入り投稿をあとで簡単に見返すことができます。  
### コメント
他社員の投稿での疑問点等を気軽にコメントで尋ねることができます。気軽に他社員とのコミュニケーションを取ることができます。  
### 検索
気になるキーワードや社員を検索することができます。
## 個人管理部分
この部分は他社員に見られることはありません。
### 数値入力
数値入力には「予算」、「売上日」、「売上」を入力することができます。
